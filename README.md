# MSG Foundation - Levantamento de Requisitos e Escopo

Este repositório contém o mapeamento de requisitos e a organização do fluxo de trabalho para o desenvolvimento do sistema da **MSG Foundation**.

## Objetivo do Projeto
O objetivo deste trabalho foi realizar a engenharia de requisitos de software, identificando as necessidades dos usuários e estruturando o product backlog utilizando metodologias ágeis.

## O que foi desenvolvido
* **User Stories:** Redação de Histórias de Usuário utilizando a estrutura padrão (*Como [ator], eu gostaria de [ação] para [objetivo]*).
* **Product Backlog:** Organização e priorização das tarefas de acordo com o perfil dos usuários (Gerente, Cliente, Administrador).
* **Gestão Ágil:** Simulação de um fluxo de trabalho utilizando um quadro Kanban.
